After clicking Copy formatted signature, paste into Gmail / Outlook / Apple Mail signature settings. The HTML source button is for signature tools that accept raw markup.

iPhone setup — signature install: iOS Mail won’t let you paste a rich signature directly in Settings. Email this signature to yourself, open the email on iPhone, long-press → Select All → Copy, paste into Settings → Mail → Signature, then (on older iOS) shake the phone and tap Undo once to keep HTML formatting.

Why the logo survives iPhone replies: iOS Mail strips every <img> element when quoting the original message, replacing it with a dash. This signature renders the logo as a CSS background-image on a clickable box (no <img>), so iOS has nothing to strip and the logo stays visible in the quoted reply. Outlook gets the same image via a VML fallback.

iPhone setup — stop the “Include Attachment?” prompt on forwards: Go to Settings → Mail → Include Attachments with Replies and pick Never (or Always). The default Ask is what triggers the prompt — iOS asks for every forward regardless of whether the image is hosted or embedded. The logo here is loaded from a hosted URL, so “Never” still leaves the logo intact for the recipient (loaded from the web).
