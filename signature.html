Where to paste it

After you click Copy signature, open your email's signature settings and paste (Ctrl+V on Windows, Cmd+V on Mac):

  • Gmail: gear icon → See all settings → General tab → scroll to Signature → Create new → paste → Save Changes at the bottom.
  • Outlook (new / web): gear icon → Account → Signatures → New signature → paste → Save.
  • Apple Mail (Mac): Mail menu → Settings → Signatures → click + → uncheck “Always match my default message font” → paste.
